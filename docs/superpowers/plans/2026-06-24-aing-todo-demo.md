# AING Todo Demo — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a reference implementation of the AING semantic-first methodology — semantic_model.yaml is the single source of truth; a generator script produces DDL, dbt tests, and Pydantic models; AI generates the FastAPI app from spec + semantic.

**Architecture:** Human writes `spec/spec.md` and `semantic/semantic_model.yaml`. Running `python generate.py` deterministically produces `generated/ddl/schema.sql`, `generated/pydantic_models.py`, and `dbt/models/schema.yml`. The FastAPI app in `app/` is generated by AI from spec + semantic — humans review the semantic model only, never the generated code.

**Tech Stack:** Python 3.12, FastAPI 0.111, psycopg2-binary 2.9, PostgreSQL 15, PyYAML 6, dbt-postgres 1.8, pytest 8, httpx 0.27

## Global Constraints

- All business rules enforced in `services/` only — routers contain no business logic
- Business rule comments in services must cite rule ID: `# BR-T01`
- `app/schemas.py` imports from `generated/pydantic_models.py` — never redefines models
- Raw SQL with psycopg2 — no ORM
- Soft deletes only — never `DELETE` on `todos` or `users`
- `generated/` and `dbt/models/schema.yml` — never edit manually; always regenerate via `python generate.py`
- Python 3.12, Pydantic v2

```markdown
# Todo App — Specification

## Domain

Application quản lý công việc cá nhân với lists, todos, và collaborators.

## Use Cases

### UC-01: User creates a todo list
- POST /lists với body {owner_id, name, is_shared?}
- name required, max 255 chars
- is_shared mặc định false

### UC-02: User adds a todo to a list
- POST /lists/{list_id}/todos với body {created_by, title, priority?, due_date?}
- title required, max 500 chars
- due_date nếu cung cấp: phải >= hôm nay (BR-T04)
- List phải chưa archive (BR-L01)

### UC-03: User updates a todo
- PATCH /todos/{id} với body {title?, status?, priority?, due_date?}
- Không thể thay đổi status hoặc title nếu todo đang completed (BR-T01)
- Khi status → completed: system tự set completed_at (BR-T02)
- Không thể đổi status nếu list đã archive (BR-T03)

### UC-04: User archives a list
- PATCH /lists/{id}/archive
- Sau archive: không thêm todo mới (BR-L01), không đổi status (BR-T03)

### UC-05: User soft-deletes a todo
- DELETE /todos/{id} → set is_deleted = true
- Không bao giờ dùng SQL DELETE

### UC-06: User views active todos của một list
- GET /lists/{list_id}/todos?active=true
- active=true filter: is_deleted=false AND status NOT IN ('completed','cancelled')

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| POST | /users | Create user |
| GET | /users/{id} | Get user |
| POST | /lists | Create list |
| GET | /lists/{id} | Get list |
| PATCH | /lists/{id}/archive | Archive list |
| POST | /lists/{list_id}/todos | Create todo |
| GET | /todos/{id} | Get todo |
| PATCH | /todos/{id} | Update todo |
| DELETE | /todos/{id} | Soft-delete todo |
| GET | /lists/{list_id}/todos | List todos (supports ?active=true) |

## Acceptance Criteria

- BR-U01, BR-U02, BR-L01, BR-L02, BR-T01, BR-T02, BR-T03, BR-T04 enforced at service layer
- completed_at never accepted as API input — system-set only (BR-T02)
- No SQL DELETE on todos or users table
- `dbt test` passes on fresh DB after schema.sql applied
- `python generate.py` runs idempotently — output identical on second run

## Out of Scope

- Authentication / JWT
- Frontend UI
- Docker Compose
- dbt CI/CD pipeline config
```

```yaml
# HUMAN-MAINTAINED — single source of truth for domain model
# Business rules have IDs (BR-xxx) — referenced by services/ in code comments
version: "1.0"
domain: TodoApp
description: "Semantic model cho ứng dụng quản lý công việc cá nhân"

entities:

  User:
    description: "Người dùng đã đăng ký"
    source_table: users
    attributes:
      id:           { type: uuid,      nullable: false }
      email:        { type: string,    nullable: false, unique: true, max_length: 255 }
      display_name: { type: string,    nullable: false, max_length: 255 }
      created_at:   { type: timestamp, nullable: false }
      is_active:
        type: boolean
        default: true
        description: "false = suspended. Không phải đã xóa — xóa là không tồn tại"
    business_rules:
      - id: BR-U01
        rule: "Email unique kể cả với tài khoản đã deactivate"
        rationale: "Tránh conflict nếu user reactivate tài khoản cũ"
      - id: BR-U02
        rule: "Không hard-delete user — chỉ set is_active = false"
        rationale: "Foreign key integrity và audit trail"
    tests:
      - unique: [email]
      - not_null: [id, email, display_name, created_at]

  TodoList:
    description: "Nhóm các todo theo chủ đề hoặc dự án"
    source_table: todo_lists
    attributes:
      id:        { type: uuid,   nullable: false }
      owner_id:  { type: uuid,   nullable: false, references: User.id }
      name:      { type: string, nullable: false, max_length: 255 }
      is_shared:
        type: boolean
        default: false
        description: "true = collaborators có thể sửa"
      archived_at:
        type: timestamp
        nullable: true
        description: "null = active. non-null = đã archive, chỉ đọc"
    business_rules:
      - id: BR-L01
        rule: "List đã archive (archived_at IS NOT NULL) không được thêm todo mới"
        rationale: "Archive = lưu lịch sử, không phải xóa"
      - id: BR-L02
        rule: "Owner không thể bị xóa khỏi collaborator list của chính mình"
    tests:
      - not_null: [id, owner_id, name]
      - custom:
          name: "no_new_todo_in_archived_list"
          sql: |
            SELECT t.id FROM todos t
            JOIN todo_lists l ON t.list_id = l.id
            WHERE l.archived_at IS NOT NULL
              AND t.created_at > l.archived_at
          expect: empty

  Todo:
    description: "Đơn vị công việc, thuộc về một list"
    source_table: todos
    attributes:
      id:         { type: uuid,   nullable: false }
      list_id:    { type: uuid,   nullable: false, references: TodoList.id }
      created_by: { type: uuid,   nullable: false, references: User.id }
      title:      { type: string, nullable: false, max_length: 500 }
      status:
        type: enum
        values: [pending, in_progress, completed, cancelled]
        default: pending
      priority:
        type: enum
        values: [low, medium, high, critical]
        default: medium
      due_date:
        type: date
        nullable: true
        description: "null = không deadline"
      completed_at:
        type: timestamp
        nullable: true
        description: "System tự set khi status → completed. Không nhận từ API input"
      is_deleted:
        type: boolean
        default: false
        description: "Soft delete — không bao giờ dùng DELETE SQL"
    business_rules:
      - id: BR-T01
        rule: "Khi status = completed, title và due_date không được phép thay đổi"
        rationale: "completed todo là bằng chứng — bảo toàn tính toàn vẹn lịch sử"
      - id: BR-T02
        rule: "completed_at chỉ được set bởi system, không phải API input"
      - id: BR-T03
        rule: "Todo trong list đã archive không được đổi status"
        depends_on: BR-L01
      - id: BR-T04
        rule: "due_date khi tạo mới không được là ngày trong quá khứ"
        exception: "Import từ hệ thống cũ: cho phép nếu is_imported = true"
    tests:
      - not_null: [id, list_id, created_by, title, status]
      - accepted_values:
          column: status
          values: [pending, in_progress, completed, cancelled]
      - custom:
          name: "completed_must_have_completed_at"
          sql: "SELECT id FROM todos WHERE status = 'completed' AND completed_at IS NULL"
          expect: empty
      - custom:
          name: "non_completed_must_not_have_completed_at"
          sql: "SELECT id FROM todos WHERE status != 'completed' AND completed_at IS NOT NULL"
          expect: empty

computed_concepts:

  OverdueTodo:
    description: "Todo quá hạn — có deadline đã qua nhưng chưa kết thúc"
    formula: >
      due_date IS NOT NULL
      AND due_date < CURRENT_DATE
      AND status NOT IN ('completed', 'cancelled')
    rationale: "cancelled không tính overdue"

  CompletionRate:
    description: "Tỷ lệ hoàn thành của một list"
    formula: "COUNT(status = 'completed') / NULLIF(COUNT(status != 'cancelled'), 0)"
    scope: per_list

  ActiveTodo:
    description: "Filter mặc định — todo đang trong vòng đời active"
    formula: "is_deleted = false AND status NOT IN ('completed', 'cancelled')"

glossary:
  "hoàn thành":
    maps_to: "status = 'completed'"
    NOT: "is_deleted = true — đó là XÓA"
  "công việc đang có":
    maps_to: computed_concept.ActiveTodo
  "quá hạn":
    maps_to: computed_concept.OverdueTodo
  "xóa":
    maps_to: "is_deleted = true"
    NOT: "SQL DELETE statement"

relationships:
  UserOwnsList:
    from: User
    to: TodoList
    type: one_to_many
    via: TodoList.owner_id
  ListContainsTodo:
    from: TodoList
    to: Todo
    type: one_to_many
    via: Todo.list_id
    semantics: "Khi list bị archive, toàn bộ todo trong đó không được đổi status"
```

```python
#!/usr/bin/env python3
"""
AING Semantic Model Generator
Usage: python generate.py
Reads: semantic/semantic_model.yaml
Writes:
  generated/ddl/schema.sql
  generated/pydantic_models.py
  dbt/models/schema.yml
"""
import sys
from pathlib import Path
import yaml

ROOT = Path(__file__).parent
SEMANTIC = ROOT / "semantic" / "semantic_model.yaml"
DDL_OUT = ROOT / "generated" / "ddl" / "schema.sql"
PYDANTIC_OUT = ROOT / "generated" / "pydantic_models.py"
DBT_OUT = ROOT / "dbt" / "models" / "schema.yml"

PG_TYPE_MAP = {
    "uuid": "UUID",
    "string": "TEXT",
    "timestamp": "TIMESTAMPTZ",
    "boolean": "BOOLEAN",
    "date": "DATE",
    "enum": "VARCHAR(50)",
    "integer": "INTEGER",
}

PY_TYPE_MAP = {
    "uuid": "UUID",
    "string": "str",
    "timestamp": "datetime",
    "boolean": "bool",
    "date": "date",
    "integer": "int",
}


def _spec(col_def) -> dict:
    """Normalize attribute: shorthand string or full dict."""
    return col_def if isinstance(col_def, dict) else {"type": col_def}


def _pg_col(name: str, spec: dict) -> str:
    col_type = spec.get("type", "string")
    max_len = spec.get("max_length")

    if col_type == "enum":
        pg_type = "VARCHAR(50)"
    elif col_type == "string" and max_len:
        pg_type = f"VARCHAR({max_len})"
    else:
        pg_type = PG_TYPE_MAP.get(col_type, "TEXT")

    null_str = " NOT NULL" if not spec.get("nullable", True) else ""

    default = spec.get("default")
    if default is None:
        default_str = ""
    elif isinstance(default, bool):
        default_str = f" DEFAULT {'true' if default else 'false'}"
    elif isinstance(default, str):
        default_str = f" DEFAULT '{default}'"
    else:
        default_str = f" DEFAULT {default}"

    return f"    {name} {pg_type}{null_str}{default_str}"


def generate_ddl(model: dict) -> str:
    lines = [
        "-- GENERATED BY generate.py — DO NOT EDIT",
        "-- Source: semantic/semantic_model.yaml",
        "",
    ]
    for entity_name, entity in model["entities"].items():
        table = entity["source_table"]
        cols, constraints = [], []

        for col_name, col_def in entity["attributes"].items():
            spec = _spec(col_def)
            cols.append(_pg_col(col_name, spec))

            if spec.get("unique"):
                constraints.append(
                    f"    CONSTRAINT uq_{table}_{col_name} UNIQUE ({col_name})"
                )
            if spec.get("type") == "enum":
                vals = ", ".join(f"'{v}'" for v in spec.get("values", []))
                constraints.append(
                    f"    CONSTRAINT chk_{table}_{col_name} CHECK ({col_name} IN ({vals}))"
                )
            ref = spec.get("references")
            if ref:
                ref_entity, ref_col = ref.split(".")
                ref_table = model["entities"][ref_entity]["source_table"]
                constraints.append(
                    f"    CONSTRAINT fk_{table}_{col_name}"
                    f" FOREIGN KEY ({col_name}) REFERENCES {ref_table}({ref_col})"
                )

        for test in entity.get("tests", []):
            if not isinstance(test, dict):
                continue
            for col_name in test.get("unique", []):
                c = f"    CONSTRAINT uq_{table}_{col_name} UNIQUE ({col_name})"
                if c not in constraints:
                    constraints.append(c)

        body = ",\n".join(cols + constraints)
        lines += [f"CREATE TABLE IF NOT EXISTS {table} (", body, ");\n"]

    return "\n".join(lines)


def generate_pydantic(model: dict) -> str:
    lines = [
        "# GENERATED BY generate.py — DO NOT EDIT",
        "# Source: semantic/semantic_model.yaml",
        "from __future__ import annotations",
        "from uuid import UUID",
        "from datetime import datetime, date",
        "from typing import Literal, Optional",
        "from pydantic import BaseModel, Field",
        "",
    ]
    for entity_name, entity in model["entities"].items():
        rule_comments = "".join(
            f"\n    # {r['id']}: {r['rule']}"
            for r in entity.get("business_rules", [])
        )
        lines.append(f"class {entity_name}(BaseModel):{rule_comments}")

        for col_name, col_def in entity["attributes"].items():
            spec = _spec(col_def)
            col_type = spec.get("type", "string")
            nullable = spec.get("nullable", True)
            default = spec.get("default")
            max_len = spec.get("max_length")

            if col_type == "enum":
                vals_str = ", ".join(f"'{v}'" for v in spec.get("values", []))
                py_type = f"Literal[{vals_str}]"
            else:
                py_type = PY_TYPE_MAP.get(col_type, "str")

            if nullable:
                py_type = f"Optional[{py_type}]"

            field_args = []
            if default is not None:
                if isinstance(default, bool):
                    field_args.append(f"default={str(default).lower()}")
                elif isinstance(default, str):
                    field_args.append(f"default='{default}'")
                else:
                    field_args.append(f"default={default}")
            elif nullable:
                field_args.append("default=None")
            if max_len:
                field_args.append(f"max_length={max_len}")

            field_str = f"Field({', '.join(field_args)})" if field_args else "..."
            lines.append(f"    {col_name}: {py_type} = {field_str}")

        lines.append("")

    return "\n".join(lines)


def generate_dbt_schema(model: dict) -> str:
    domain = model.get("domain", "app").lower()
    tables = []

    for entity_name, entity in model["entities"].items():
        table_name = entity["source_table"]
        col_tests: dict[str, list] = {}

        for test in entity.get("tests", []):
            if not isinstance(test, dict):
                continue
            for c in test.get("not_null", []):
                col_tests.setdefault(c, []).append("not_null")
            for c in test.get("unique", []):
                col_tests.setdefault(c, []).append("unique")

        columns = []
        for col_name, col_def in entity["attributes"].items():
            spec = _spec(col_def)
            tests = list(col_tests.get(col_name, []))

            if not spec.get("nullable", True) and "not_null" not in tests:
                tests.append("not_null")
            if spec.get("unique") and "unique" not in tests:
                tests.append("unique")
            if spec.get("type") == "enum":
                tests.append({"accepted_values": {"values": spec.get("values", [])}})
            ref = spec.get("references")
            if ref:
                ref_entity, ref_col = ref.split(".")
                ref_table = model["entities"][ref_entity]["source_table"]
                tests.append({
                    "relationships": {
                        "to": f"source('{domain}', '{ref_table}')",
                        "field": ref_col,
                    }
                })

            if tests:
                columns.append({"name": col_name, "tests": tests})

        table_entry: dict = {"name": table_name, "columns": columns}

        custom_tests = [
            t["custom"]
            for t in entity.get("tests", [])
            if isinstance(t, dict) and "custom" in t
        ]
        if custom_tests:
            table_entry["tests"] = [
                {ct["name"]: {"sql": ct["sql"].strip()}} for ct in custom_tests
            ]

        tables.append(table_entry)

    output = {
        "version": 2,
        "sources": [{"name": domain, "schema": "public", "tables": tables}],
    }
    return yaml.dump(output, default_flow_style=False, sort_keys=False, allow_unicode=True)


def main() -> None:
    if not SEMANTIC.exists():
        print(f"ERROR: {SEMANTIC} not found", file=sys.stderr)
        sys.exit(1)

    model = yaml.safe_load(SEMANTIC.read_text(encoding="utf-8"))

    DDL_OUT.parent.mkdir(parents=True, exist_ok=True)
    DDL_OUT.write_text(generate_ddl(model), encoding="utf-8")
    print(f"✓ {DDL_OUT}")

    PYDANTIC_OUT.parent.mkdir(parents=True, exist_ok=True)
    PYDANTIC_OUT.write_text(generate_pydantic(model), encoding="utf-8")
    print(f"✓ {PYDANTIC_OUT}")

    DBT_OUT.parent.mkdir(parents=True, exist_ok=True)
    DBT_OUT.write_text(generate_dbt_schema(model), encoding="utf-8")
    print(f"✓ {DBT_OUT}")


if __name__ == "__main__":
    main()
```

```python
# app/services/list_service.py
from datetime import datetime, timezone
import psycopg2.extensions


def create_list(conn: psycopg2.extensions.connection, owner_id: str, name: str, is_shared: bool = False) -> dict:
    with conn.cursor() as cur:
        cur.execute(
            """
            INSERT INTO todo_lists (id, owner_id, name, is_shared)
            VALUES (gen_random_uuid(), %s, %s, %s)
            RETURNING id, owner_id, name, is_shared, archived_at
            """,
            (owner_id, name, is_shared),
        )
        row = cur.fetchone()
        conn.commit()
        return _row_to_list(row)


def get_list(conn: psycopg2.extensions.connection, list_id: str) -> dict | None:
    with conn.cursor() as cur:
        cur.execute(
            "SELECT id, owner_id, name, is_shared, archived_at FROM todo_lists WHERE id = %s",
            (list_id,),
        )
        row = cur.fetchone()
        return _row_to_list(row) if row else None


def archive_list(conn: psycopg2.extensions.connection, list_id: str) -> dict | None:
    # BR-L01: after archive, no new todos allowed (enforced in todo_service)
    with conn.cursor() as cur:
        cur.execute(
            """
            UPDATE todo_lists SET archived_at = %s
            WHERE id = %s AND archived_at IS NULL
            RETURNING id, owner_id, name, is_shared, archived_at
            """,
            (datetime.now(timezone.utc), list_id),
        )
        row = cur.fetchone()
        conn.commit()
        return _row_to_list(row) if row else None


def _row_to_list(row: tuple) -> dict:
    return {
        "id": str(row[0]),
        "owner_id": str(row[1]),
        "name": row[2],
        "is_shared": row[3],
        "archived_at": row[4].isoformat() if row[4] else None,
    }
```

```python
def create_user(body: CreateUserRequest) -> dict:
    conn = get_conn()
    try:
        with conn.cursor() as cur:
            # BR-U01: email unique across all accounts
            cur.execute("SELECT id FROM users WHERE email = %s", (body.email,))
            if cur.fetchone():
                raise HTTPException(409, "Email already registered (BR-U01)")
            cur.execute(
                """
                INSERT INTO users (id, email, display_name, created_at, is_active)
                VALUES (gen_random_uuid(), %s, %s, NOW(), true)
                RETURNING id, email, display_name, created_at, is_active
                """,
                (body.email, body.display_name),
            )
            row = cur.fetchone()
            conn.commit()
            return {
                "id": str(row[0]),
                "email": row[1],
                "display_name": row[2],
                "created_at": row[3].isoformat(),
                "is_active": row[4],
            }
    finally:
        put_conn(conn)
```

```python
# app/services/todo_service.py
from datetime import datetime, date, timezone
import psycopg2.extensions


def _get_todo(cur, todo_id: str) -> tuple | None:
    cur.execute(
        """
        SELECT t.id, t.list_id, t.created_by, t.title, t.status,
               t.priority, t.due_date, t.completed_at, t.is_deleted,
               l.archived_at
        FROM todos t
        JOIN todo_lists l ON t.list_id = l.id
        WHERE t.id = %s
        """,
        (todo_id,),
    )
    return cur.fetchone()


def _row_to_todo(row: tuple) -> dict:
    return {
        "id": str(row[0]),
        "list_id": str(row[1]),
        "created_by": str(row[2]),
        "title": row[3],
        "status": row[4],
        "priority": row[5],
        "due_date": row[6].isoformat() if row[6] else None,
        "completed_at": row[7].isoformat() if row[7] else None,
        "is_deleted": row[8],
    }


def create_todo(
    conn: psycopg2.extensions.connection,
    list_id: str,
    created_by: str,
    title: str,
    priority: str = "medium",
    due_date: date | None = None,
    is_imported: bool = False,
) -> dict:
    with conn.cursor() as cur:
        # BR-L01: cannot add todo to archived list
        cur.execute("SELECT archived_at FROM todo_lists WHERE id = %s", (list_id,))
        row = cur.fetchone()
        if not row:
            raise ValueError("List not found")
        if row[0] is not None:
            raise PermissionError("Cannot add todo to archived list (BR-L01)")

        # BR-T04: due_date must not be in the past
        if due_date and due_date < date.today() and not is_imported:
            raise ValueError("due_date cannot be in the past (BR-T04)")

        cur.execute(
            """
            INSERT INTO todos (id, list_id, created_by, title, status, priority, due_date, is_deleted)
            VALUES (gen_random_uuid(), %s, %s, %s, 'pending', %s, %s, false)
            RETURNING id, list_id, created_by, title, status, priority, due_date, completed_at, is_deleted
            """,
            (list_id, created_by, title, priority, due_date),
        )
        row = cur.fetchone()
        conn.commit()
        # Append None for archived_at to match _row_to_todo signature
        return _row_to_todo(row + (None,))


def get_todo(conn: psycopg2.extensions.connection, todo_id: str) -> dict | None:
    with conn.cursor() as cur:
        row = _get_todo(cur, todo_id)
        return _row_to_todo(row) if row else None


def update_todo(
    conn: psycopg2.extensions.connection,
    todo_id: str,
    title: str | None = None,
    status: str | None = None,
    priority: str | None = None,
    due_date: date | None = None,
) -> dict:
    with conn.cursor() as cur:
        row = _get_todo(cur, todo_id)
        if not row:
            raise ValueError("Todo not found")

        current_status = row[4]
        list_archived_at = row[9]

        # BR-T01: completed todo cannot change title or status
        if current_status == "completed" and (title is not None or status is not None):
            raise PermissionError(
                "Cannot modify completed todo title or status (BR-T01)"
            )

        # BR-T03: todo in archived list cannot change status
        if list_archived_at is not None and status is not None:
            raise PermissionError(
                "Cannot change status of todo in archived list (BR-T03)"
            )

        updates = {}
        if title is not None:
            updates["title"] = title
        if status is not None:
            updates["status"] = status
        if priority is not None:
            updates["priority"] = priority
        if due_date is not None:
            updates["due_date"] = due_date

        # BR-T02: system sets completed_at when status → completed
        if status == "completed":
            updates["completed_at"] = datetime.now(timezone.utc)

        if not updates:
            return _row_to_todo(row)

        set_clause = ", ".join(f"{k} = %s" for k in updates)
        values = list(updates.values()) + [todo_id]
        cur.execute(
            f"""
            UPDATE todos SET {set_clause}
            WHERE id = %s
            RETURNING id, list_id, created_by, title, status, priority,
                      due_date, completed_at, is_deleted
            """,
            values,
        )
        updated = cur.fetchone()
        conn.commit()
        return _row_to_todo(updated + (None,))


def soft_delete_todo(conn: psycopg2.extensions.connection, todo_id: str) -> dict | None:
    with conn.cursor() as cur:
        cur.execute(
            """
            UPDATE todos SET is_deleted = true
            WHERE id = %s
            RETURNING id, list_id, created_by, title, status, priority,
                      due_date, completed_at, is_deleted
            """,
            (todo_id,),
        )
        row = cur.fetchone()
        conn.commit()
        return _row_to_todo(row + (None,)) if row else None


def list_todos(
    conn: psycopg2.extensions.connection, list_id: str, active_only: bool = False
) -> list[dict]:
    with conn.cursor() as cur:
        if active_only:
            # ActiveTodo computed concept from semantic model
            cur.execute(
                """
                SELECT id, list_id, created_by, title, status, priority,
                       due_date, completed_at, is_deleted
                FROM todos
                WHERE list_id = %s
                  AND is_deleted = false
                  AND status NOT IN ('completed', 'cancelled')
                """,
                (list_id,),
            )
        else:
            cur.execute(
                """
                SELECT id, list_id, created_by, title, status, priority,
                       due_date, completed_at, is_deleted
                FROM todos WHERE list_id = %s
                """,
                (list_id,),
            )
        return [_row_to_todo(r + (None,)) for r in cur.fetchall()]
```

```python
# app/routers/todos.py
from datetime import date
from fastapi import APIRouter, HTTPException, Query
from pydantic import BaseModel, Field
from app.db import get_conn, put_conn
from app.services import todo_service

router = APIRouter(tags=["todos"])


class CreateTodoRequest(BaseModel):
    created_by: str
    title: str = Field(max_length=500)
    priority: str = "medium"
    due_date: date | None = None


class UpdateTodoRequest(BaseModel):
    title: str | None = Field(default=None, max_length=500)
    status: str | None = None
    priority: str | None = None
    due_date: date | None = None


@router.post("/lists/{list_id}/todos", status_code=201)
def create_todo(list_id: str, body: CreateTodoRequest) -> dict:
    conn = get_conn()
    try:
        try:
            return todo_service.create_todo(
                conn, list_id, body.created_by, body.title, body.priority, body.due_date
            )
        except PermissionError as e:
            raise HTTPException(409, str(e))
        except ValueError as e:
            raise HTTPException(404, str(e))
    finally:
        put_conn(conn)


@router.get("/todos/{todo_id}")
def get_todo(todo_id: str) -> dict:
    conn = get_conn()
    try:
        result = todo_service.get_todo(conn, todo_id)
        if not result:
            raise HTTPException(404, "Todo not found")
        return result
    finally:
        put_conn(conn)


@router.patch("/todos/{todo_id}")
def update_todo(todo_id: str, body: UpdateTodoRequest) -> dict:
    conn = get_conn()
    try:
        try:
            return todo_service.update_todo(
                conn, todo_id, body.title, body.status, body.priority, body.due_date
            )
        except PermissionError as e:
            raise HTTPException(409, str(e))
        except ValueError as e:
            raise HTTPException(404, str(e))
    finally:
        put_conn(conn)


@router.delete("/todos/{todo_id}")
def delete_todo(todo_id: str) -> dict:
    conn = get_conn()
    try:
        result = todo_service.soft_delete_todo(conn, todo_id)
        if not result:
            raise HTTPException(404, "Todo not found")
        return result
    finally:
        put_conn(conn)


@router.get("/lists/{list_id}/todos")
def list_todos(list_id: str, active: bool = Query(default=False)) -> list:
    conn = get_conn()
    try:
        return todo_service.list_todos(conn, list_id, active_only=active)
    finally:
        put_conn(conn)
```
